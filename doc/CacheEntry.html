createCache(cacheExists, callback, errorCallback)

Creates a cache in storage. If storage is not available, calls an error. If storage is WebSQL, update instead of insert. If content is css, recursively look for url and replace it with absolute url (and hopefully replace with cached url in the future).

Parameters:
Name Type Description
cacheExists Boolean

true if cache already exists in stroage. otherwise false.
callback Function

Called when the storage successfully store the content.
errorCallback Function

Called when the storage failed to store the content.
